### 文档命名规范

文档采用统一命名格式：`{序号}-DeepDiary_{文档名称}.md`

- **序号**：01-08，便于排序
- **项目名**：DeepDiary，保持项目名称原样
- **文档名**：使用下划线分隔，清晰明了
